Start-AI 项目旨在提供一个 **从零基础到进阶** 学习人工智能的高质量资料库，包括人工智能理论和实践，深度学习框架（TensorFlow，Keras），机器学习云平台等。

希望不论是人工智能从业者，亦或是有志于投身AI的新同学，都能够勇敢踏出第一步，并且持之以恒地学习。
